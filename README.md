# BNUOJ-Code-Similarity-Detect
针对BNUOJ的代码查重辅助工具

BNU Online Judge:

校外：http://www.bnuoj.com

校内：http://acm.bnu.edu.cn

## 可用于：
* 比赛查重，IP定位
* 作业查重

## 功能：
* 抓取IP对应的网络环境，由此判定交题时的终端物理位置合法性
* 删除注释片段并重新格式化代码
* 通过Levenshtein距离来判断代码相似度
